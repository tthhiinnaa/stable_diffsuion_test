# 사용법

## flask
- webUI가 실행되고 있는 서버에 세팅 해야 한다.
- base로 사용할 이미지를 동일한 dir에 "base.png"로 저장해야 한다. 


```bash

# 가상환경 실행 후
pip install -r requirement.txt
# flask 실행
python main.py

```


## Gradio

- flask 를 실행한 서버와 private ip 를 통해 통신이 가능한 서버에 설치 해야 한다. 


```bash

# 가상환경 실행 후
pip install -r requirement.txt
# Gradio 실행
python main.py

```
